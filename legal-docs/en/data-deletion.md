# Data Deletion Instructions - CodeWithBotina

## How to delete your data

If you want to delete your account and all associated data from CodeWithBotina, follow these steps:

### Option 1: Automatic Deletion (Coming Soon)

1. Sign in at https://blog.codewithbotina.com
2. Go to Account Settings
3. Click "Delete my account"
4. Confirm deletion

### Option 2: Manual Request (Current Method)

Send an email to **support@codewithbotina.com** with:

**Subject:** Data deletion request

**Message:**
```
Name: [Your name]
Email: [The email you used to sign in]
Request: Complete deletion of my account and data

I confirm that I understand this action is irreversible.
```

### Data that will be deleted

- Profile information (name, email, photo)
- All comments
- All reactions (like/dislike)
- Session history
- Authentication tokens

### Data we cannot delete

- Anonymous server logs (required by law for 90 days)
- Automatic backups (deleted in the next purge cycle)

### Processing time

- Immediate: account data and sessions
- Within 24 hours: comments and reactions
- Within 30 days: backups and logs

You will receive a confirmation email when deletion is complete.

### Revoke Google OAuth access

You can also revoke access from Google:

1. Visit https://myaccount.google.com/permissions
2. Find "CodeWithBotina Blog"
3. Click "Remove access"
